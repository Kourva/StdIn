# StdIn
Simple note app (for coding) in Dark Theme | Python Kivy 

It works with a TextInput inside a ScrollView
Text font in **RobotoMono-Regular**

# Install
> for running on linux
+ clone
```bash
git clone https://github.com/Kourva/StdIn && cd StdIn
```
+ requirements
```bash
pip install kivy
```
+ run
```bash
python3 main.py
```
